# Smart Resume Screening Tool

An AI-powered ATS Resume Analyzer built using Flask and Machine Learning.

## Features

- Resume Upload (PDF)
- Resume Parsing
- ATS Score
- Resume Strength Rating
- TF-IDF Similarity
- Skill Matching
- Missing Skills Detection
- Resume Completeness Analysis
- Recommendations
- PDF Report Generation
- Interactive Dashboard
- Chart.js Visualization
- Dark Mode
- Responsive UI

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-learn
- PyPDF2
- ReportLab
- Chart.js

## Installation

```bash
pip install -r requirements.txt
python app.py
```
